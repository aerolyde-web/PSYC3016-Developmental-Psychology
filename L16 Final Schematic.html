L16: Infant Cognition II - Nativism vs Constructivism

PSYC3016 Final Schematic | Theorists: Carey, Spelke, Baillargeon (Nativist) vs Cohen, Cashon (Constructivist)
Core Thesis: NOT "nature vs nurture" - both sides agree genes & learning matter. The REAL debate: domain-specific modules with core knowledge vs domain-general statistical learning. Where is development located - input systems or knowledge itself?

The Real Debate

QuestionNativistConstructivist
What's innate?Domain-specific modules with core knowledge (objects persist, solidity)Domain-general info processing detecting low-level features (color, motion, edges)
Triggers?Specific perceptual inputs activate modules (e.g., common motion triggers object module)Statistical regularities gradually build representations through experience

Exam Trap: Early ability ≠ innateness. 4 months = billions of visual experiences. Rapid learning IS possible!

Kellman & Spelke (1983): Object Unity

Baillargeon (1987): Drawbridge/VoE

Method: Familiarize to 180° rotation. Place box behind. Test: Possible (stops ~120°, perceptually novel) vs Impossible (full 180° through box, perceptually familiar).
Result: 4.5mo look longer at impossible event
Nativist: Conceptual surprise overrides perceptual familiarity = core knowledge of permanence + solidity
Cohen & Cashon Critique:
  • Included infants who didn't meet 50% habituation criterion
  • Per Hunter & Ames: incomplete habituation → familiarity preference
  • 180° has more motion/complexity → perceptual confound

How to Argue Both Sides

NATIVIST Argument Template:
  1. Identify specific perceptual trigger (common motion)
  2. Show ability emerges when trigger perceivable
  3. Demonstrate co-emergence of related abilities
  4. Attribute gradual changes to input maturation
Carey's Co-Emergence: Unity + solidity + individuation all ~4mo = unified module activating. Constructivism has no reason for simultaneity.
CONSTRUCTIVIST Argument Template:
  1. Document gradual improvement (newborn→2mo→4mo)
  2. Show graded effects (more support needed younger)
  3. Demonstrate dissociations (feature memory before spatial)
  4. Attribute changes to representation construction
Statistical Learning: 4mo = ~150M visual samples. Co-motion predicts unity 99%+ in nature = learnable regularity.
