L23-24: Nativism vs Constructivism — Point-by-Point Debate

Each row = one debate issue with directly opposing studies

Debate Issue NATIVIST Position CONSTRUCTIONIST Counter
Structural Dependence
Is structure-dependent syntax innate or learned from statistics?
Crain & Nakayama (1987)
0 errors / 600+ questions
Children (3-5yo) never moved first auxiliary (*"Is the boy who smoking is crazy?"). Always moved main-clause "is" despite linear rule being simpler + consistent with input.
→ POINT: Zero errors = innate constraint. Children never CONSIDER linear rule because UG prohibits structure-independent operations. "Parade case" for innateness.
Ambridge, Rowland & Pine (2008)
"is": 0% errors | "can": 7% errors
SD errors do occur with "can" questions (7%) but not "is" (0%). Why? "Who smoking" = ungrammatical (TP≈0); "who smoke" = grammatical bigram elsewhere.
→ COUNTER: Word-specific variation contradicts universal constraint. Errors occur WHERE STATISTICS SUPPORT THEM. Structural dependence is emergent, not innate.
Input Quality
Is linguistic input impoverished or informationally rich?
Poverty of Stimulus (Chomsky)
No negative evidence
Children acquire constraints (island constraints on wh-movement) that cannot be learned from input. No one produces ungrammatical sentences to show what's prohibited.
→ POINT: If knowledge is unlearnable from experience, it must be INNATE. Input is too impoverished to support acquisition without UG.
Saffran et al. (1996) + Mintz (2003)
TP: 1.0 vs 0.33 | Frames: 87%
Saffran: 8-month-olds segment speech using transitional probabilities (2 min exposure).
Mintz: Frequent frames ("the ___ is") predict categories with 87% accuracy.
→ COUNTER: Input is RICH, not impoverished. Domain-general statistical learning extracts structure — TP for segmentation, distributional context for categories.
Abstraction Timing
Is syntax abstract from the start or gradually constructed?
Gertner, Fisher & Eisengart (2006)
21-month-olds succeed
21-month-olds hearing "The frog is gorping the bear" (novel verb) looked longer at agent-patient scene. Mapped transitive syntax → semantics without knowing verb.
→ POINT: Verb-GENERAL abstract syntax by 21 months — near onset of multi-word speech. NOT item-specific verb islands.
Tomasello & Brooks (1998)
2;0: 19% → 2;6: 44% → 4yr: 90%
Novel verb trained intransitive ("sock tamming"), tested transitive. Only 19% of 2-year-olds extended — rises to 44% at 2;6, ~90% by age 4.
→ COUNTER: Age-graded trajectory = GRADUAL abstraction. Early syntax is verb-specific islands ("I kick X"). Abstract schemas built via analogical comparison.
Priming Confound
Does early "abstraction" require exemplar priming?
Fisher et al. (2006) — Original
21mo + transitive warm-up = SUCCESS
Warm-up trials used transitive frames with same nouns as test: "The frog is washing the bear." Then tested novel verb "gorping" — children succeeded.
→ CLAIM: 21-month-olds have abstract transitive syntax (NP1-V-NP2 → agent-patient). Production conservatism masks early competence.
Dittmar et al. (2008) — Replication
− transitive warm-up = FAILURE
Same paradigm BUT replaced transitive warm-ups with generic descriptions ("This is called washing"). 21-month-olds FAILED — no agent-patient preference.
→ DECISIVE COUNTER: Fisher's "abstract syntax" depends on recent exemplar priming. Knowledge is ITEM-SPECIFIC ([FROG][VERB][BEAR]), not verb-general.
Sensitive Period
How do children exceed their input?
Senghas, Kita & Özyürek (2004) — NSL
Cohort 1: 25% → Cohort 2: 75% componential
NSL emerged in 1980s Nicaragua. Cohort 2 children learned FROM Cohort 1 adults but produced MORE linguistic structure (75% vs 25% componential). Children EXCEEDED input during sensitive period.
NATIVIST: UG imposes compositional structure on deficient input during maturational window. Innate language faculty drives restructuring.
CONSTRUCTIONIST: Domain-general pattern-extraction + iterated learning. General cognitive biases optimize communication across generations — no UG needed.
