1 The Great Debate: Nativism vs. Constructivism

This is THE central tension across all 5 lectures. Every finding can be framed through this lens.

L14: Classic Theories
  • Chomsky (Nativist): Poverty of stimulus → UG innate
  • Piaget (Constructivist): Sensorimotor → symbols built
  • Vygotsky (Cultural): Symbols transmitted, not constructed alone
L16: Object Knowledge
  • Spelke/Baillargeon: Core knowledge modules at 4mo
  • Cohen/Cashon: Methodological critiques (familiarity pref?)
  • Meltzoff & Moore: Piecemeal development (features before space)
L17: Dynamic Systems
  • Thelen/Smith: Reject both! No modules, no symbols
  • Embodiment permanent: Never transcended
  • Statistical learning: Domain-general mechanism
L18: Relational Learning
  • Leslie: Innate causal module (6mo shapes)
  • Oakes & Cohen: Load-dependent, not modular
  • Neoconstructivism: 5 principles of hierarchical construction

The Evolution of the Debate Across Lectures

L14: Classic positions established (Chomsky vs. Piaget)

L16: Empirical tests - VoE paradigm becomes battleground

L17: Third position emerges - Dynamic Systems rejects both representationalism AND nativism

L18: Neoconstructivism synthesizes - domain-general hierarchical construction with capacity constraints

2 Cognitive Overload → Reversion to Lower-Level Processing

THE signature prediction of constructivism. This pattern appears in L15, L17, and L18 - use it to connect studies!

L15: Methods
  • Hunter & Ames (1988): Under-habituated → familiarity preference
  • Non-monotonic curve: encoding phase mimics novelty detection
  • Key insight: Processing state determines preference direction
L17: Statistical Learning
  • Younger & Cohen Exp 4: 7mo with EXTENDED time on medium task
  • Result: REVERTED to features (pattern matches 4mo!)
  • Key insight: More time can produce WORSE outcomes
L18: Causal Perception
  • Oakes & Cohen (1990): 6mo shapes ✓, complex toys ✗
  • Cohen & Oakes (1993): 10mo repeated ✓, varying ✗
  • Ferry et al.: Familiar-not-habituated objects → failure
WHY THIS MATTERS: Nativism predicts modules fire CONSISTENTLY when triggered. The fact that SAME AGE infants show OPPOSITE results based on load CANNOT be explained by module maturation. Only hierarchical construction with capacity-dependent access predicts this pattern.

The Formula (Apply Across All Studies)

Relational/Conceptual Processing = f(Available Capacity - Task Load)

Where Load = Object Complexity + Object Variability + Object Salience

When Load exceeds threshold → system reverts to featural/perceptual encoding

SAQ Power Move

Connect Younger & Cohen (L17) to Oakes & Cohen (L18): "The same pattern appears in feature correlation learning (Younger & Cohen, 1986) and causal perception (Oakes & Cohen, 1990) - when cognitive load exceeds capacity, infants strategically shift to lower-level processing. This cross-domain consistency supports domain-GENERAL hierarchical construction over domain-SPECIFIC modules."

3 Hierarchical Construction: Features → Patterns → Relations

The building blocks principle. Higher levels are built ON TOP OF lower levels - you can't skip steps.

L15: Pattern Extraction
  • Cohen & Strauss (1979):
  • Condition I (1 face, 1 angle) → narrow encoding
  • Condition II (1 face, many angles) → view-invariant identity
  • Condition III (many faces, many angles) → category abstraction
  • Principle: Habituation variability = generalization scope
L16: Object Knowledge
  • Meltzoff & Moore (1998):
  • 5mo: Feature memory ✓, spatial continuity ✗
  • 9mo: Both ✓
  • Principle: Features (Level 1) before space (Level 2)
L17: Brain Architecture
  • Level 1: Modality-specific cortex (features)
  • Level 2: ATL hub (cross-modal correlations)
  • Level 3: aPFC (relations among relations)
  • Principle: Increasingly abstract statistical regularities
L18: Relational Learning
  • Same/Different: Need multiple exemplars to force abstraction beyond object features
  • Causal Perception: Object encoding must complete before causal relation accessible
  • Principle: Hierarchical dependency - incomplete lower level blocks higher

4 The Competence-Performance Gap: Looking vs. Doing

The methodological divide. Why do infants "know" things at 4mo (looking) but can't "use" them until 9mo (reaching)?

L14: Piaget's Timeline
  • Object permanence: Stage 4 (8-12mo) via manual search
  • A-not-B error persists until ~12mo
  • Piaget's view: Knowledge develops gradually through action
L16: The Dissociation
  • Looking-time success: 4.5mo (Baillargeon VoE)
  • Manual search success: 8-9mo (Piaget)
  • ~4 month gap!
L17: A-not-B Reinterpreted
  • Linda Smith (1999): Postural reset eliminates error
  • Error drops: 80% → 20-30%
  • Not permanence failure: Embodied motor memory!
TWO INTERPRETATIONS:
Nativist: Knowledge exists early (COMPETENCE) but motor/executive functions immature (PERFORMANCE). Looking taps knowledge; reaching requires motor planning, inhibition.

Constructivist: True permanence develops at 8-9mo; earlier looking = shallow perceptual memory, not conceptual knowledge. Looking and reaching tap DIFFERENT representations.

SAQ Application

Use Linda Smith's A-not-B postural reset to bridge L14 and L17: "Piaget interpreted the A-not-B error as incomplete object permanence, but Dynamic Systems theorists (Smith, 1999) demonstrated that simply changing the infant's posture between trials eliminates the error (80% → 20-30%). This suggests the error reflects embodied motor memory rather than conceptual limitation, challenging both Piaget's constructivism AND Spelke's nativism by proposing cognition remains permanently embodied."

5 Embodiment: Does Abstraction Ever Transcend the Body?

L14's Stage 6 vs. L17's Dynamic Systems. A fundamental disagreement about the endpoint of development.

L14: Piaget's View
  • Sensorimotor foundation → Symbolic operations
  • Stage 6 (18-24mo): Mental combinations emerge
  • Symbols TRANSCEND sensorimotor origins
  • Endpoint: Abstract, decontextualized thought
L17: Dynamic Systems View
  • Embodiment is PERMANENT
  • Apparent abstraction = complex integration, not transcendence
  • Adults show sensory-motor activation during "abstract" reasoning (fMRI)
  • Endpoint: Symbols don't exist! Only embodied patterns
L17: Evidence for Permanent Embodiment
  • Sticky Mittens: Motor change → perceptual change
  • A-not-B Postural Reset: Body state = memory state
  • Gap Crossing: Spatial knowledge is posture-specific
  • Thelen Twins: No cross-task transfer
  • Action Words (fMRI): "Kick" activates leg motor cortex
  • Color Semantics: "Canary" activates yellow visual cortex

The Fundamental Question

Piaget: How do discrete SYMBOLS emerge from continuous sensorimotor experience? (The "symbol problem")

Dynamic Systems: They DON'T. The question assumes symbols exist. Instead, there's only increasingly complex patterns of distributed sensory-motor activation. ATL integrates patterns; aPFC detects meta-patterns. No amodal semantics required.

6 Statistical Learning as Alternative to Modules

The domain-general mechanism. Can pattern extraction from experience explain what nativists attribute to innate modules?

L15: Pattern Extraction Basics
  • Cohen & Strauss: Infants extract invariants across varied exemplars
  • Emerges ~30 weeks (developmental milestone)
  • Principle: What varies becomes irrelevant; what's constant becomes the pattern
L16: Experience Argument
  • 150 MILLION visual samples by 4 months
  • Co-motion predicts unity 99%+ in natural environment
  • Counter to "too early": Massive experience available
L17: Shape Bias
  • Correlates with vocabulary SIZE (r=.72), not age (r=.45)
  • Emerges at 25-50 noun threshold
  • IS TRAINABLE
  • Proves: Learned meta-pattern, not innate constraint
L18: Same/Different
  • 1 pair repeated → encode object identity (insufficient variability)
  • 4 pairs → force relational abstraction
  • Principle: Optimal variability extracts relation, not instances

The Nativist Counter-Argument

Chomsky/Spelke: Sheer quantity doesn't matter if mechanism is domain-general. Can't extract ABSTRACT principles (solidity, binding constraints) from low-level correlations. Statistical learning can detect patterns but can't explain WHY certain patterns are privileged.

Constructivist Response: ATL and aPFC provide hierarchical integration that transforms low-level correlations into increasingly abstract representations. No qualitative jump to "symbols" needed—just quantitative increases in integration complexity.

The Master Synthesis: How It All Connects

For ANY Cognitive Development SAQ, You Can Use This Structure:

  1. State the theoretical tension (Nativist modules vs. Constructivist hierarchical learning)
  2. Give nativist evidence (Kellman & Spelke, Baillargeon, Leslie - early competence)
  3. Give constructivist counter-evidence (Cohen's critiques, Oakes & Cohen load effects, Meltzoff & Moore dissociation)
  4. Cite the signature prediction: Cognitive overload → reversion to lower-level processing
  5. Connect to Dynamic Systems (embodiment, piecemeal development) as third position
  6. End with methodological nuance (habituation vs. familiarization, competence-performance gap)

The Three Most Powerful Cross-Lecture Connections:

1. Oakes & Cohen (L18) + Younger & Cohen (L17): Same load-dependent pattern in different domains (causal perception + feature correlation) → supports domain-GENERAL mechanism

2. Meltzoff & Moore (L16) + Cohen & Strauss (L15): Piecemeal development (features before space) + hierarchical construction (variability determines abstraction level) → same principle

3. A-not-B (L14) reinterpreted by Smith (L17): Classic Piagetian error explained by embodiment → bridges classic and modern perspectives
