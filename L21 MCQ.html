Q1
Executive function is BEST defined as:
A) The ability to remember information
B) Top-down cognitive control processes that regulate thoughts, emotions, and behaviors in service of goals

D) Language ability
üí° Explanation:
Executive function refers to the "executive" control processes that manage and regulate other cognitive systems‚Äîlike a CEO directing a company. These processes control:

Q2
The three primary subcomponents of executive function in developmental research are:
A) Reading, writing, arithmetic
B) Memory, attention, language
C) Response inhibition, working memory, cognitive flexibility
D) Vision, hearing, touch
üí° Explanation:
The three core EF components (sometimes called "cool" EF):
  • Response Inhibition: Suppressing prepotent/automatic responses
  • Working Memory: Holding and manipulating information
  • Cognitive Flexibility: Switching between tasks/rules/mental sets
These three are highly correlated but separable (factor analysis shows distinct factors). All improve from ages 3-5, continue through adolescence, peak in mid-20s.

Q8
Response inhibition refers to:
A) Being shy or withdrawn
B) Suppressing prepotent (automatic/habitual) responses to do something else instead
C) Not responding at all
D) Responding slowly
üí° Explanation:
Response inhibition = impulse control: Overriding the most alluring, automatic, or habitual response.

Examples:
  • See "RED" in green letters ‚Üí want to say "red" (reading is automatic) ‚Üí inhibit to say "green"
  • See sun ‚Üí want to say "day" ‚Üí inhibit to say "night" (day-night task)
  • See marshmallow ‚Üí want to eat now ‚Üí inhibit to wait for two marshmallows
  • Colleague invites you to pub ‚Üí want to go ‚Üí inhibit to finish paper
Most related to self-control, delay of gratification, impulse control.

Q9
The classic Stroop task measures response inhibition by:
A) Asking participants to read words quickly
B) Requiring participants to name the ink color while inhibiting automatic word-reading (e.g., "RED" in green ink ‚Üí say "green")
C) Testing color blindness
D) Measuring reading speed
üí° Explanation:
The Stroop task creates conflict between:
  • Automatic process: Reading the word (overlearned, fast, automatic)
  • Task requirement: Naming ink color (requires controlled attention)

Problem for children: Can't read yet! So researchers use adapted versions.

Q21
Halford's relational complexity theory proposes that working memory capacity is best characterized by:
A) Just the total number of items you can remember
B) The number of elements you can consider IN PARALLEL while processing their interrelationships
C) How fast you can memorize
D) How long you can remember things
üí° Explanation:
Halford's key insight: WM limitation isn't about NUMBER of items‚Äîit's about how many things you must consider TOGETHER (in parallel) because they RELATE to each other.

If items are independent, you can "chunk" and serialize (think one at a time). But when items INTERACT, you must hold them simultaneously to understand relationships. Adult limit: ~4 elements in parallel (explaining why 4-way statistical interactions are uninterpretable!).
üìä Example: Understanding a 3-way interaction (Fresh/Frozen √ó Chocolate/Carrot √ó Fred's/Bob's bakery) taxes WM because all variables must be considered together.
Halford Relational Complexity Parallel Processing CRITICAL
