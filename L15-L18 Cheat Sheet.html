1. HABITUATION PARADIGM: Core Mechanics (L15)

Standard Criterion: 3 consecutive trials below 50% of first-3-trial baseline
Example: Trials 1-3: 18s, 16s, 20s → Baseline = 18s → Threshold = 9s → Need 3 trials ALL < 9s
Dishabituation = recovery of looking → discrimination. Maintained habituation = no recovery → generalization OR cannot discriminate.
  • Two-Phase Architecture: (1) Habituation phase: repeated exposure until criterion → (2) Test phase: novel vs familiar comparison. Both required—test alone meaningless without knowing habituation state.
  • Procedural Controls: Caregiver wears occluding headphones (prevents social cueing); observer blinded to stimulus (sees only infant's face); attention-getter between trials standardizes trial initiation.
  • Available DVs: Looking time (default—available birth, tolerates movement), sucking rate (birth–6mo), reaching (~4-5mo, confounded by motor), heart rate deceleration (attentional engagement). Match measure to question!
  • Most dangerous error: Including infants who never reached criterion—they enter test in unpredictable preference states, contaminating data.
  • Cohen & Strauss (1979): Generalization paradigm—vary habituation breadth to control abstraction level. Cond 1 (1 face, 1 pose): viewpoint-specific. Cond 2 (1 face, multi-pose): identity-invariant. Cond 3 (multi-face, multi-pose): category "faces." Breadth→generalization level.
Criterion = 3 trials < 50% baseline. Dishabituation = discrimination. Cohen & Strauss: habituation breadth controls abstraction level. Never include non-habituators!

2. HUNTER & AMES (1988): Non-Monotonic Model (L15)

  • U-Shaped Function: Preference vs exposure is NON-MONOTONIC. Brief → familiarity preference (active encoding, incomplete consolidation). Extended → novelty preference (full encoding complete, familiar boring). Null (no pref) = transition point, UNINTERPRETABLE.
  • Moderators: (1) Stimulus complexity: complex stimuli require longer encoding → extended familiarity phase. (2) Infant age: younger = slower processing → longer familiarity phase. 4mo needs more trials than 8mo for same stimulus.
  • Familiarity Preference = CAN discriminate (otherwise no preference); familiarization insufficient. NOT paradoxical—infant preferentially attends to complete encoding of partially-processed stimulus. ≠ "prefers familiar things."
  • Null Result Ambiguity: No preference could mean: (a) at transition point, (b) cannot discriminate, (c) over-habituated/fatigued. NEVER conclude "cannot discriminate" from null alone—must diagnose preference state.
U-curve: brief→familiarity pref, extended→novelty pref, null=ambiguous. Familiarity pref = CAN discriminate but under-habituated. Null is uninterpretable without diagnostics. Complexity & age moderate curve position.

3. NATIVISM vs CONSTRUCTIVISM: Core Debate (L16)

QuestionNativist (Spelke, Carey, Baillargeon)Constructivist (Cohen, Cashon)
What's innate?Domain-SPECIFIC modules with core knowledge (objects persist, solids don't interpenetrate)Domain-GENERAL processes detecting features, statistical regularities
Trigger?Specific perceptual inputs activate modules (common motion triggers object module)Statistical regularities gradually build representations
Age change?Perceptual prerequisites mature → pre-existing knowledge expressed (competence precedes performance)Sufficient experience accumulated → representation constructed
  • NOT "nature vs nurture"—both agree genes AND learning matter. Real debate: domain-specific vs domain-general learning mechanisms. Frame correctly or lose marks!
  • Nativist signature: Co-emergence of related abilities (object unity, solidity, individuation all ~4mo = unified module). Constructivist signature: Piecemeal development, graded improvement, dissociations.
  • Kellman & Spelke (1983): 4mo habituated to rod moving behind occluder → look longer at broken rod than complete (conceptual surprise overrides perceptual novelty). Stationary condition = no preference → common motion is specific trigger.
  • Johnson & Aslin (1998): 2mo need ~70% visible rod; 4mo succeed with ~40%. Slater (1996): Newborns need minimal occlusion. Gradual improvement = constructivist signature.
NOT nature/nurture—domain-specific vs domain-general. Nativist: modules with core knowledge, specific triggers (common motion). Constructivist: statistical learning, piecemeal development. Kellman & Spelke: broken > complete (4mo moving); stationary = no pref.

Impossible: 180° through box (familiar). 4.5mo look longer at impossible.Nativist: Core knowledge of permanence + solidity. Cohen critique: Included non-habituators → familiarity preference confound.

VoE: longer looking to impossible = core knowledge? BUT Cohen critique: familiarity preference if non-habituators included. Meltzoff: 5mo feature memory ✓, spatial continuity ✗; 9mo both ✓ = DISSOCIATION vindicates Piaget.

5. DYNAMIC SYSTEMS & EMBODIED COGNITION (L17)

  • Core Principle: Development driven by organism-task-environment interaction, not brain alone. Intelligent behavior EMERGES from brain-body-world coupling. All cognition embodied. Distinguishes from Piaget (stages) and nativist (maturation).
  • Piecemeal Development: Skills context-specific; minimal generalization across postures. Thelen twins: Intensive roller-skating (Johnny) → expert skating but NO acceleration of sitting/crawling/walking vs untrained twin (Jimmy). Motor learning specific to trained context only.
  • Visual Cliff: New crawlers (6mo) cross deep side without fear; experienced crawlers (9mo) refuse; new walkers must RELEARN. Risk perception embodied in posture.
  • Gap Crossing (Adolph): 9mo experienced sitters refuse dangerous gaps; SAME infants attempt identical gaps when crawling. Zero generalization sitting→crawling.
  • Smith (1999) A-not-B: Having infant stand up and sit down between A-trials and B-trial ELIMINATES perseverative error. Object representation is body-state-dependent—not incomplete permanence but embodied motor habit.
  • Sticky Mittens: 2mo given Velcro mittens → 6mo-level visual inspection and oral exploration. Motor ability causally drives perceptual development.
  • Embodied Semantics (fMRI): "kick"→leg motor cortex; "canary"→yellow visual cortex. Word meanings ARE distributed sensory-motor activations (constitutive). Integration hubs: ATL (all modalities), aPFC/BA10 (relations-among-relations, analogy).
Dynamic systems = organism-task-environment interaction. Piecemeal: sitting ≠ crawling ≠ walking. Smith A-not-B: postural reset eliminates error → embodied motor habit, not incomplete permanence. Embodied semantics: "kick"→leg motor cortex.

6. YOUNGER & COHEN: Feature Correlation Learning (L17)

Test Pattern Interpretation:
C low, U low, N high = Features only (both C and U familiar because features seen before)
C low, U high, N high = Correlations learned (U novel because violates learned structure)
Flat habituation = Attempting correlations but overloaded (struggling, not disengaged)
StudyTaskResult & Interpretation
Y&C 1983 HARD4 features, 2 pairs correlated10mo: correlations. 4/7mo: features only. Age-dependent capacity.
Y&C 1986 E1 EASY3 features, ALL correlated7mo: correlations ✓. 4mo: features. Proves 7mo CAN do correlations when simplified.
Y&C 1986 E2 MED3 features, 2/3 correlated7mo: FLAT habituation (attempting but overloaded). 10mo: correlations. 4mo: features.
Y&C 1986 E4 MED+3 features, 2/3 + extended time7mo: REVERTED to features (same as 4mo!). More time = WORSE when exceeds capacity.
  • Constructivist Processing Principle: Infants use highest level possible; if overloaded, revert to lower level while still incorporating information. Explains non-monotonic performance.
  • Shape Bias (Gershkoff-Stowe 2004): Correlates with vocabulary SIZE not age. <25 nouns: ~30%, 26-50: ~60-70%, 51+: ~90%+. Proves bias is LEARNED, not innate.
  • Domain-Specific Weighting: Not universal—artifacts=shape; natural kinds=shape+texture; mass nouns=material. Children sensitive to count/mass syntax by 24mo.
7mo transitional: EASY=correlations, MED=flat (struggling), MED+extended=REVERTS to features. More time = WORSE if overloaded. Shape bias: vocabulary size (not age) → learned. <25 nouns: 30%; 51+: 90%+.

7. ABSTRACT RELATIONAL LEARNING: Same/Different & Causal (L18)

StudyAgeFinding & Interpretation
Ferry et al. (2015)7-9mo4 diverse pairs → same/different discrimination ✓. 1 pair = encode object not relation. Familiar-not-habituated = FAILURE (salience interference).
Anderson (2018)3mo2 pairs → success ✓. 6 pairs → FAILURE (overload). 3mo humans > adult chimps in relational matching!
Leslie (1984)6moSimple shapes: causal perception ✓ (categorize by causal status). Nativist: innate causal module.
Oakes & Cohen (1990)6mo/10moComplex toys: 6mo FAIL, 10mo ✓. Same spatiotemporal cues! Object complexity consumes capacity.
Cohen & Oakes (1993)10moComplex repeated: causal ✓. Complex variable: REVERT to featural. Variability exceeds capacity.
  • Natural Partitions (Gentner 1982): Objects = stable, direct perception, LOW demand. Relations = dynamic, require abstraction across instances, HIGH demand. Relations built ON TOP of objects.
  • Object Salience Effect: Novel objects → success ✓. Habituated objects → success ✓. Familiar-but-NOT-habituated → FAILURE (unexpected re-encounter increases salience, blocks relational access).
  • Human Uniqueness: 3mo human infants abstract same/different with 2 pairs. Adult chimps need extensive symbol training. Relational thought distinguishes humans.
  • Younger = novelty to everything: In generalization studies, younger infants show novelty preference to ALL test stimuli = features only processing (not better discrimination!). Pattern develops with capacity.
Age-specific optimal exemplars: 3mo=2 pairs, 7-9mo=4 pairs. Familiar-not-habituated = FAILURE (salience). Leslie 6mo causal with simple shapes; Oakes & Cohen: 6mo FAIL with complex toys, 10mo variable = REVERT. Neoconstructivism explains all.

8. COGNITIVE OVERLOAD: Neoconstructivist Signature (L18)

Load FactorLow Load → RELATIONALHigh Load → FEATURAL
Object ComplexitySimple shapes → 6mo causal ✓ (Leslie)Complex toys → 6mo featural (O&C)
Object VariabilityRepeated objects → 10mo causal ✓Variable objects → 10mo featural (C&O)
Exemplar Count2 pairs (3mo) / 4 pairs (7-9mo)6 pairs (3mo) = overload; 1 pair = object
Object SalienceNovel OR habituated → successFamiliar-not-habituated → failure
  • Neoconstructivism Signature: Increasing cognitive load → lower information processing level. Same infant succeeds/fails depending on load even when relational structure identical.
  • Why Nativism Fails: If module triggered by specific input, should fire regardless of object properties. But 6mo causal with simple shapes ≠ 6mo with complex toys. Same spatiotemporal cues! No nativist mechanism for this.
  • Hierarchical Construction: Objects encoded FIRST (lower level); relations built FROM patterns across objects (higher level). When object processing exceeds capacity, relational processing becomes inaccessible.
  • Formula: Relational success = f(Capacityage − Loadcognitive) where Load = complexity + variability + exemplar count + salience.
Signature pattern: load ↑ → processing level ↓. Same infant, same relation, different outcome based on load. Nativism can't explain task-dependent variation. Neoconstructivism: objects first, relations built on top—capacity-dependent.
