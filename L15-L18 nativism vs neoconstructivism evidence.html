üß¨ NATIVIST (Carey, Spelke, Baillargeon)

  • Modules: Specialized cognitive circuits for objects, language, number
  • Core knowledge: Objects persist when occluded; solid objects can't interpenetrate
  • Specific triggers: Common motion ‚Üí object module; contact + immediate motion ‚Üí causal module
  • Development: Maturation of perceptual prerequisites allows modules to apply innate knowledge
  • Anti-Piagetian: Object permanence present at 4mo, not developed at 8-9mo
  • Prediction: Abilities co-emerge when module "turns on"; robust to stimulus variation

üîß NEOCONSTRUCTIVIST (Cohen, Cashon, Oakes)

  • No innate knowledge: Only domain-general processing systems
  • Low-level features: Detect color, motion, edges innately
  • Hierarchical construction: Build higher-level units from lower-level relationships
  • Statistical regularities: Co-moving surfaces ‚Üí same object (learned heuristic)
  • Development: Gradual incremental construction of representations
  • Neo-Piagetian: Support Piaget's 8-9mo timeline for true permanence
  • Prediction: Load-dependent; success with simple/fail with complex (same relational structure)
üéØ THE DECISIVE TEST: If modules exist, they should fire when triggering input is present regardless of stimulus complexity. If hierarchical construction is correct, increasing cognitive load should cause systematic reversion to lower-level processing‚Äîeven when relational structure is held constant. This is the signature pattern nativism cannot explain.

[THESIS] While nativist core knowledge accounts explain early competence in looking-time paradigms, neoconstructivist capacity-limited accounts better explain the systematic load-dependent pattern where infants succeed with simple stimuli but fail with complex stimuli‚Äîeven when relational structure is held constant.

[OBJECT KNOWLEDGE] Nativists cite Kellman & Spelke (1983): 4-month-olds perceive object unity when rod parts show common motion (80% vs 50% stationary), and Baillargeon (1987): 4.5-month-olds look longer at impossible drawbridge events (d ‚âà 0.7-0.9), suggesting core knowledge of persistence and solidity. However, constructivists counter with Johnson & Aslin (1998): 2-month-olds succeed only with small occluders‚Äîgradual development, not modular emergence‚Äîand Meltzoff & Moore (1998): 5-month-olds track features but fail spatial continuity until 9 months, supporting Piaget's timeline.

[RELATIONAL LEARNING - DECISIVE] The causal perception literature provides decisive evidence. Leslie (1984) found 6-month-olds categorize by causal status with simple geometric shapes. However, Oakes & Cohen (1990) showed the same 6-month-olds revert to featural processing with complex toys‚Äîdespite identical spatiotemporal structure. If a causal module existed, it should fire regardless of object complexity. By 10 months, causal perception is restored with complex repeated objects, but Cohen & Oakes (1993) showed even 10-month-olds revert to featural when complex objects vary trial-by-trial. This systematic pattern‚Äîsame age, same relational structure, different outcomes based solely on cognitive load‚Äîcannot be explained by modules.

[SAME/DIFFERENT] Anderson et al. (2018) demonstrated 3-month-olds abstract same/different with 2 pairs (p < .05) but fail with 6 pairs‚Äîa non-monotonic pattern where too few exemplars yield object encoding and too many cause overload.

[SYNTHESIS] The neoconstructivist formula‚ÄîRelational Success = f(Capacityage ‚àí Loadcognitive)‚Äîunifies all findings. Early competence exists but is fragile, hierarchically dependent on object processing, and subject to overload. This graded, task-dependent pattern is the signature prediction nativism cannot accommodate.
