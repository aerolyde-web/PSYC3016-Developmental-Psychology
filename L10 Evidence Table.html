Basolateral Amygdala (BLA) Function Neurobiological model CU traits neurobiology BLA functions: automatic attention allocation to social cues, encoding specific stimulus features, fast prediction error, gaze direction control BLA = underactive in CU traits. Functions include: automatic attention allocation (detecting important social info without conscious effort), encoding specific features (fine discriminations like $1 vs $5, fearful vs sad expression), fast prediction error (rapid learning from violations), gaze direction control (shifting attention to eyes). Evolutionarily recent mammalian structure supporting flexible adaptive learning. Clinical signature: poor automatic attention to social cues. BLA activation: 0.2 in CU vs 0.8 in typical development (75% reduction in automatic activation) Explains impaired implicit/automatic social learning while explicit/conscious processing remains intact; core deficit in CU traits "BLA underactive in CU (0.2 vs 0.8): impairs automatic attention to social cues, fine discrimination, fast learning - flexible adaptive learning."
Central Amygdala (CeA) Function Neurobiological model CU traits neurobiology CeA functions: explicit emotion recognition, general valence processing, physiological fear response, slow updating CeA = normal or elevated in CU traits. Functions include: explicit emotion recognition (conscious identification when attention directed), general valence processing (broad positive vs negative), physiological fear response generation (startle, heart rate, cortisol), slow updating (stable emotional learning). Evolutionarily ancient "lizard brain" shared across vertebrates. Clinical signature: normal performance when explicitly directed. CeA activation normal or elevated in CU traits; intact when attention explicitly directed Explains preserved explicit/conscious emotion processing and why explicit instruction eliminates deficits; bypasses BLA impairment "CeA normal in CU: intact explicit emotion recognition, general valence, fear response, slow learning - ancient survival system."
Differential Amygdala Dissociation Synthesized neurobiological model Explains task-dependent CU performance BLA underactivation + CeA normal activation explains pattern where deficits present in implicit tasks but disappear with explicit instruction Dissociation: Intact explicit processing coexists with impaired implicit processing.
